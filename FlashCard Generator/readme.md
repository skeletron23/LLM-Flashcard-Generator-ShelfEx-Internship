# LLM-Powered Flashcard Generator

[cite_start]This project is an LLM-powered Flashcard Generation Tool developed as an internship assignment for ShelfEx.  [cite_start]It is designed to convert educational content into effective question-answer flashcards. 

## Project Objectives
* [cite_start]Ingest educational materials (e.g., textbook excerpts, lecture notes). 
* [cite_start]Automatically extract and generate relevant flashcards (Q&A format). 
* [cite_start]Offer a basic UI/UX for interaction, viewing, and exporting cards. 

## Core Technologies
* [cite_start]**Language:** Python 
* [cite_start]**Framework/UI:** Streamlit 
* [cite_start]**LLM Interface:** OpenAI GPT (gpt-3.5-turbo) 

## Setup (Local Development)

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/YOUR_GITHUB_USERNAME/LLM-Flashcard-Generator-ShelfEx-Internship.git](https://github.com/YOUR_GITHUB_USERNAME/LLM-Flashcard-Generator-ShelfEx-Internship.git)
    cd LLM-Flashcard-Generator-ShelfEx-Internship
    ```
    (Replace `YOUR_GITHUB_USERNAME` with your actual GitHub username.)

2.  **Create a virtual environment (recommended):**
    ```bash
    python -m venv venv
    # On Windows:
    .\venv\Scripts\activate
    # On macOS/Linux:
    source venv/bin/activate
    ```

3.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Set up your OpenAI API Key:**
    Create a file named `.env` in the root directory of the project and add your OpenAI API key:
    ```
    OPENAI_API_KEY="sk-your_openai_api_key_here"
    ```
    **Remember to replace `"sk-your_openai_api_key_here"` with your actual key.**
